# Handwritten-digit-recognition-CNN
CNN model for handwritten digit classification using the MNIST dataset

Project Workflow
1)Load Dataset – Load the MNIST handwritten digit dataset.
2)Preprocess Data – Normalize pixel values and reshape images for CNN input.
3)Build Models – Create CNN and MLP models using TensorFlow/Keras.
4)Train Models – Train both models with early stopping to prevent overfitting.
5)Evaluate Performance – Compare CNN and MLP accuracy on the test dataset.
6)Digit Prediction – Predict handwritten digits using the trained CNN model.
7)Feature Visualization – Display CNN feature maps from convolutional layers.
8)Interactive Canvas – Draw a digit on a canvas and get real-time predictions.
